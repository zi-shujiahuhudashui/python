## 代码运行指南

### 环境要求

- Python 3.7 或更高版本
- 依赖库：`numpy`, `matplotlib`, `pillow`

### 内容

- A（车联网）（一维三车）.py和A（车联网）（二维三车）.py两个主程序，分别为一维三车编队控制仿真和二维三车编队控制仿真
- 运行后生成的结果图片和动图存放在目录（代码中可自动创建）

### 运行步骤

1. 打开终端（或 PyCharm 终端），进入代码所在目录

2. 运行主程序：

   bash

   ```python
   A（车联网）（一维三车）.py
   ```

   或

   ```python
   A（车联网）（二维三车）.py
   ```

3. 程序会自动进行仿真，并显示三张静态图（轨迹、误差、速度），同时在当前目录（或指定目录）生成一个 `formation.gif` 动图文件

### 切换通信拓扑

要复现论文中的两种通信拓扑，只需修改代码开头的邻接矩阵 `A`：

- **Case I（全连通）**：当前使用默认的全连通矩阵（所有车互相通信）
- **Case II（断连）**：改为断连矩阵（领导与跟随不通信，跟随之间通信）。注释有断连矩阵，使用代码中注释的 `A` 矩阵即可

修改 `A` 后重新运行可得到对应结果

### 结果说明

运行完成后，你会得到以下文件：

- `Fig4_trajectories.png`：车辆二维轨迹图
- `Fig5_errors.png`：纵向和横向误差图
- `Fig6_velocities.png`：x、y 方向速度曲线
- `formation.gif`：编队过程的动态演示

如果切换拓扑，建议手动重命名结果文件，或修改代码中的保存路径以避免覆盖。
