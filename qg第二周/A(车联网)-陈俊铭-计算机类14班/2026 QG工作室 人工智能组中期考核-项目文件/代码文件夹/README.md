## 车联网编队控制仿真（选题 A）

## 项目简介

本项目复现了论文《Feedback-based platoon control for connected autonomous vehicles under different communication network topologies》中的核心控制协议（公式 (3)），实现了二维三车的编队控制仿真，并支持自适应结束和动图输出。

## 依赖环境

* Python 3.7+

- numpy
- matplotlib
- pillow（用于生成 GIF）
- pycharm

## 包含两个代码

一个是A（车联网）（一维三车），一个是A（车联网）（二维三车）

## 切换说明

修改代码开头的 `A`（邻接矩阵）以切换拓扑：

- Case I（全连通）：使用代码中注释的 `A` 矩阵使用当前示例矩阵
- Case II（断连）：使用代码中注释的 `A` 矩阵

## 自适应结束

程序会自动判断编队是否完成（所有跟随车与期望位置的欧氏距离小于 0.05 m 且持续 1 秒），并提前结束仿真，打印完成时间。

## 结果说明

- 在两种通信拓扑下，车辆均能形成期望编队（车1在领导后方2米，车2在领导右后方2米）。
- 断连拓扑下收敛速度明显变慢，与论文结论一致。
